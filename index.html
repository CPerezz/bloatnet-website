Bloatnet Initiative

Advancing Ethereum's scalability through experimental blockchain parameters

Bloatnet is an ambitious effort to create a new Ethereum-like chain with altered parameters that enable greater throughput through bigger gas limits and smaller slot times.

Core Objectives

1

Critical State Growth Monitoring

Know when state growth becomes a critical issue → prioritize statelessness

Understand the exact thresholds where state bloat begins to impact network performance and client synchronization. This critical monitoring helps us prioritize statelessness implementation before the network experiences malfunctions.

2

Massive State Testnet Data

Gather extensive performance data with high gas limits

Collect comprehensive performance metrics in a controlled testnet environment with significantly higher gas limits. This data provides insights into network behavior under extreme load conditions.

3

Path to EIP-7938

Understand when and how clients will struggle with state growth

Clear the implementation path for EIP-7938, which proposes incremental gas-limit upgrades over time. By identifying obstacles and safe thresholds, we ensure smooth implementation of this critical scalability improvement.
